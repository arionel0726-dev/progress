# 🗺️ Roadmap Tracker

A clean, visual tool for tracking progress on your roadmap — milestones, goals, and tasks all in one place.

🔗 **Live:** [https://progress-nw64.vercel.app/](https://progress-nw64.vercel.app/)

---

## ✨ Features

- 📌 **Visual roadmap** — see all your goals and milestones at a glance
- ✅ **Progress tracking** — mark tasks and steps as done
- 🗂️ **Organized structure** — group items by phases or categories
- ⚡ **Fast & lightweight** — no unnecessary bloat, just what you need

---

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm or yarn

### Installation

```bash
git clone https://github.com/your-username/roadmap-tracker.git
cd roadmap-tracker
npm install
```

### Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for production

```bash
npm run build
npm start
```

---

## 🌐 Deployment

This project is deployed on **Vercel**. To deploy your own instance:

1. Fork the repo
2. Import it into [Vercel](https://vercel.com/)
3. Click **Deploy** — that's it
